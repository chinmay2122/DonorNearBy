## Step 1: Supabase Database Setup ⚙️

Run this SQL in your Supabase SQL Editor:

```sql
-- Blood Banks Table
create table if not exists public.blood_banks (
  id bigint generated by default as identity primary key,
  name text not null,
  city text not null,
  address text not null,
  phone text not null,
  lat double precision not null,
  lng double precision not null,
  inventory jsonb not null default '{}'::jsonb,
  created_at timestamptz default now()
);

-- Requests Table
create table if not exists public.requests (
  id bigint generated by default as identity primary key,
  patient_name text not null,
  hospital_name text not null,
  blood_type text not null,
  units integer not null check (units > 0),
  city text not null,
  contact_number text not null,
  status text not null default 'pending',
  notes text,
  created_at timestamptz not null default now()
);

-- Donations Table
create table if not exists public.donations (
  id bigint generated by default as identity primary key,
  donor_name text not null,
  blood_type text not null,
  units integer not null check (units > 0),
  city text not null,
  contact_number text not null,
  availability_date date,
  status text not null default 'pending',
  notes text,
  created_at timestamptz not null default now()
);

-- Donors Table (NEW)
create table if not exists public.donors (
  id bigint generated by default as identity primary key,
  name text not null,
  email text unique not null,
  phone text not null,
  blood_type text not null,
  age integer,
  city text not null,
  address text,
  lat double precision not null,
  lng double precision not null,
  last_donation_date date,
  status text not null default 'pending',
  approval_date timestamptz,
  rejection_reason text,
  created_at timestamptz not null default now(),
  updated_at timestamptz not null default now()
);

-- Admin Users Table (NEW)
create table if not exists public.admin_users (
  id bigint generated by default as identity primary key,
  username text unique not null,
  password text not null,
  email text unique not null,
  full_name text,
  role text not null default 'admin',
  is_active boolean default true,
  created_at timestamptz not null default now()
);

-- Approval Logs Table (NEW)
create table if not exists public.approval_logs (
  id bigint generated by default as identity primary key,
  donor_id bigint not null references public.donors(id) on delete cascade,
  admin_id bigint not null references public.admin_users(id) on delete cascade,
  action text not null,
  reason text,
  created_at timestamptz not null default now()
);
```

## Step 2: Backend Setup 🔧

```bash
cd backend

# Install dependencies
npm install

# Create .env file
cat > .env << EOF
SUPABASE_URL=your_supabase_url_here
SUPABASE_KEY=your_supabase_anon_key_here
PORT=5000
EOF

# Start development server
npm run dev
```

You should see:
```
🩸 Blood Bank API running on http://localhost:5000
🔒 Admin Credentials (Demo):
   Username: admin
   Password: <set-in-your-env>
📍 Donor Registration: POST /api/donors/register
🔍 Search Nearby Donors: GET /api/donors/search?blood_type=B+&lat=28.6315&lng=77.2167&radius=10
```

## Step 3: Frontend Setup 🎨

```bash
cd frontend

# Install dependencies
npm install

# Create .env file
echo "REACT_APP_API_URL=http://localhost:5000/api" > .env

# Start React app
npm start
```

App opens at `http://localhost:3000`

## 🆘 Troubleshooting

### Port 5000 already in use?
```bash
# Change PORT in backend/.env
PORT=5001

# Update frontend/.env
REACT_APP_API_URL=http://localhost:5001/api
```

### CORS errors?
- Backend CORS is enabled
- Check frontend API_URL matches backend port
- Restart both servers

### Geolocation not working?
- Use localhost (not 127.0.0.1)
- Click popup to grant permission
- Check browser console for specifics

### Admin login fails?
- Username: `admin` (lowercase)
- Password: `<set-in-your-env>`
- Check .env has SUPABASE_URL and SUPABASE_KEY

### Search returns no results?
- Sample approved donors are in database
- Try blood type: B+
- Use coordinates in Delhi area (lat: 28.6315, lng: 77.2167)
